What is an API request?

One request means one software call to Calibre’s data endpoint — for example: fetch one player profile, one comparison, one watchlist, or one rankings board.

Scout allowance

1,500 requests/month gives a serious scout dashboard room to work without inviting abuse or server bills we cannot control. That is roughly 50 requests/day before paid add-ons.

Club contracts

Higher limits are negotiated by use case, rate limit, caching, redistribution rights and third-party data costs.

Do not sell raw third-party data as if Calibre owns it. Calibre should sell intelligence, workflow, ratings layers, watchlists, comparisons and approved derived views.
